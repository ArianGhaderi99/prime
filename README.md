# Prime

A fast prime number checker with CSV-based caching, designed to improve performance when working with large numbers and repeated calculations.

## ⚡ Features

* Prime number detection
* CSV-based caching system
* Optimized for large numbers
* Faster repeated lookups
* Lightweight and easy to use

## 🛠 Built With

* Python
* CSV

## 📂 About

This project implements a prime number checking system that stores previously computed results in a CSV cache. By reusing stored values, the application significantly reduces computation time for repeated prime checks, especially when dealing with large numbers.

## 🚀 Use Cases

* Mathematical experiments
* Number theory projects
* Performance optimization studies
* Large number prime validation
